# To-Do App

A simple python application to manage daily tasks efficiently.

## Project Overview
This project allows users to create, view, update, and delete tasks. It is designed to help users organize their work or personal tasks in an easy-to-use interface.

## Features
- Add new tasks with title  
- Mark tasks as completed  
- Delete tasks when done  
- View all tasks in a simple interface

## Technologies Used
- Python
